│  │         │  │         │  │         │  │         │        │
│  ├─────────┤  ├─────────┤  ├─────────┤  ├─────────┤        │

**컴포넌트:**

| 컴포넌트 | 설명 |
|----------|------|
| 정렬 드롭다운 | 최신순 / 인기순 |
| 만들기 버튼 | 로그인 시에만 표시 |
| 카드 그리드 | 4열 (반응형: 태블릿 2열, 모바일 1열) |
| 무한스크롤 | Intersection Observer 사용 |

**카드 컴포넌트:**
```

```

│                                                             │
│  [X 닫기]                                                   │

│         ┌───────────────────────────────────┐              │

│                                                      [▲]   │
│         ┌──────┐                                    [♥]   │
